Column Description
1 This column will start at the first day of the month to the last day of the month.
2 This column shows the beginning O.R. number of the day.
3 This column shows the ending O.R. number of the day.
4 This column shows the Grand Total Accumulated Gross Sales ending balance of the day. - It is the grand total gross sales from day 1 to the end of this business day.
5 This column shows the Grand Total Accumulated Gross Sales beginning balance of the day. - It is the grand total gross sales from day 1 to the previous business day.
6 This column represents the Total Gross Sales for the day. This amount can be counter checked also by subtracting the ending balance with the beginning balance of the day of the Grand Total Accumulated Gross Sales.
7 This column has a default value of zero as we do not use this column because our system does not provide voiding transactions.
8 This column shows the Gross Sales from POS for the day, it is in this case, the same with column 6.
9 This column represents the VATable Sales for the day.
10 This column represents the VAT Amount, it is based on the amount of Gross Sales of the day. Please take note that any discount given to the parker can not be detected from the Gross Sales in the computation of VAT Amount.
11 This column represents the VAT Exempt Sales for the day.
12 This column has the default value of zero as we do not use this column because parker can not claim zero rated incentive on the parking fee.
13 This column represents the Regular Discount given to the parker. Please take note that the VAT Amount portion on the discount shall not be included in the discount per BIR.
14 This column represent the Special Discount given to the Senior Citizen / PWD. Please take note that even it is free of charge to Senior Citizen/PWD, the transaction shall be treated as a 100% discount from the gross amount.
15 This column has the default value of zero as we do not use this column because the system has no Void/Cancelled/Returns transactions.
16 This column has the default value of zero as we do not use this column because the system has no Void/Cancelled/Returns transactions.
17 This column is the sum total of columns 13 to 16, it represents the total sales discounts given, this sales discounts amount shall be deducted from the gross sales to arrive at the Net Sales in the income tax report.
18 This column represents the VAT on Special Discount allowed by the law given to the Senior Citizen / PWD.
19 This column has the default value of zero as we do not use this column because the system has no Void/Cancelled/Returns transactions.
20 This column has the default value of zero as we do not use this column because the system has no Other Adjustments on VAT
21 This column represents the Total Adjustments or Deductions on VAT which is allowed by the law.
22 This column represents the total VAT Amount Payable to the BIR. This amount can be counter checked also by subtracting column 10 with column 21.
23 This column represents the Net Sales Amount for the day. Please take note that this Net Sales AMount iw without VAT. This amount can be counter check also by subtracting column 8 with column 17 and column 10.
24 This column has the default value of zero as we do not have any other income besides parking fee.
25 This column has the default value of zero as the system records exact amount of fees.
26 This column represets the Total Net Sales Amount for the day, it is the sum of columns 23 to 25.
27 This column has the default vakue of zero as the system has no reset counter because it provides more that enough digits for the O.R. number.
28 This column is for you to put on any remarks on the report of transactions for the day.
